## 判定标准

### 老牌白名单判定

- **入选条件**：开业时间早于 2023 年，且 2026 数据源仍可检索
- **降权条件**：总分低于 60、速度或延迟低于 40、官网不可识别、频道/群组缺失、评价出现稳定性下降
- **结论方式**：老牌不等于无风险，只表示运营时间信号强；仍需月付试用和近期状态复核

### 跑路风险判定

- **高风险信号**：强推长期套餐、官网频繁换域名、频道停更、群组刷屏、支付只剩不可逆方式、评价出现封号或大面积超时
- **榜单处理**：疑似跑路或强防雷条目不进入公开榜单；仓库只保留判定标准，不展示跑路名单
- **支付方式说明**：原始 API 未稳定披露支付方式，本仓库不伪造支付结论；支付方式只作为人工复核项
