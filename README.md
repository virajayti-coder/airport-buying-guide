## 筛选逻辑

- **怕死（防雷/跑路类）**：优先排除低分、官网缺失、频道缺失、强 AFF 噪声、用户评价含“封号/速度极差/解锁稀碎/稳定性下降”的机场
- **没钱（预算/学生党类）**：优先看 20 元以内、100G 以上、月付可试、无强年付暗示、用户评价不出现大面积晚高峰崩盘
- **要干活（AI/流媒体类）**：优先看稳定和延迟分，线路类型倾向专线/中转/多入口，避免纯低价直连且解锁地区异常的条目
- **要稳定（高端/专线类）**：优先看总分 80 以上、稳定分 80 以上、线路为专线或多入口优化、官网与频道可追踪

## 判定标准

### 老牌白名单判定

- **入选条件**：开业时间早于 2023 年，且 2026 数据源仍可检索
- **降权条件**：总分低于 60、速度或延迟低于 40、官网不可识别、频道/群组缺失、评价出现稳定性下降
- **结论方式**：老牌不等于无风险，只表示运营时间信号强；仍需月付试用和近期状态复核

### 跑路风险判定

- **高风险信号**：强推长期套餐、官网频繁换域名、频道停更、群组刷屏、支付只剩不可逆方式、评价出现封号或大面积超时
- **中风险信号**：新开不足 6 个月、官网短链/跳转链、价格显著低于同类、单运营商入口、节点数量过少
- **低风险信号**：开业时间可核、频道持续更新、官网直达、月付可试、总分和稳定分长期维持高位
- **支付方式说明**：原始 API 未稳定披露支付方式，本仓库不伪造支付结论；支付方式只作为人工复核项
